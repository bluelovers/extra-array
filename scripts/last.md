Get last value of [array].

```javascript
const lastOf = require('@extra-array/last');
// lastOf(<array>, [index=-1])

lastOf(['j', 'o', 'h', 'n']);
// 'n'
lastOf(['l', 'i', 'z'], -2);
// 'i'
```
