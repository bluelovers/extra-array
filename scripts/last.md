Get last value of [array].

```javascript
const last = require('@extra-array/last');
// last(<array>, [index=-1])

last(['j', 'o', 'h', 'n']);
// 'n'
last(['l', 'i', 'z'], -2);
// 'i'
```
> With [extra-array] try `Array.last()` instead.
