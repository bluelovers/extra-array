Get average of values in [array].

```javascript
const average = require('@extra-array/average');
// average(<array>, [begin=0], [end])

average([4, 1, 2, 3]);
// 2.5
average([4, 1, 2, 3], 1);
// 2
average([4, 1, 2, 3], 1, 3);
// 1.5
```
> With [extra-array] try `Array.average()` instead.
