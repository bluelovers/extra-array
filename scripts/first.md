Get first value of [array].

```javascript
const first = require('@extra-array/first');
// first(<array>, [index=0])

first(['g', 'a', 'r', 'f', 'i', 'e', 'l', 'd']);
// 'g'
first(['o', 'd', 'i', 'e'], 2);
// 'i'
```
> With [extra-array] try `Array.first()` instead.
