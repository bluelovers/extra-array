Get sum of values in [array].

```javascript
const sumOf = require('@extra-array/sum');
// sumOf(<array>, [begin=0], [end])

sumOf([7, 1, 9]);
// 17
sumOf([7, 1, 9], 1);
// 10
sumOf([7, 1, 9], 1, 2);
// 1
```
