Get sum of values in [array].

```javascript
const sum = require('@extra-array/sum');
// sum(<array>, [begin=0], [end])

sum([7, 1, 9]);
// 17
sum([7, 1, 9], 1);
// 10
sum([7, 1, 9], 1, 2);
// 1
```
> With [extra-array] try `Array.sum()` instead.
