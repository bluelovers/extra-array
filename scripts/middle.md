Get middle value of [array].

```javascript
const middle = require('@extra-array/middle');
// middle(<array>, [index=0])

middle(['m', 'a', 'r', 'a', 'm', 'e', 'l']);
// 'm'
middle(['m', 'i', 'l', 'k', 'm', 'a', 'n'], 1);
// 'i'
middle(['l', 'u', 'k', 'a'], -1);
// 'a'
```
> With [extra-array] try `Array.middle()` instead.
